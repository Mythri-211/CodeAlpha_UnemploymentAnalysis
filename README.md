# 🚗 Car Price Prediction using Machine Learning

## Project Description
This project predicts car prices using Machine Learning algorithms.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Machine Learning Model
Random Forest Regressor

## Features Used
- Fuel Type
- Transmission
- Present Price
- Kms Driven
- Car Age

## Model Performance
The model was evaluated using appropriate metrics and achieved good prediction accuracy.

## Output
The model predicts selling prices of cars based on input features.

## Conclusion
The project demonstrates how machine learning can be used to predict car prices based on key features.
